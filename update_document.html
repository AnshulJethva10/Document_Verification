About Document Updates

When you update a document:

  • A new hash is calculated for the updated document
  • A new blockchain transaction is created
  • Both versions will remain verifiable
  • The update history is stored on the blockchain

Important Note

Updates are permanent and will create a new blockchain entry. The original document will remain in the blockchain's history.
